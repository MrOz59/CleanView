# Shorts Clean View

Moves the YouTube Shorts metapanel into a clean side dock so the video stays unobstructed.

**Features**
- Side dock for the Shorts metapanel
- Auto-fit to screen size (no scroll in the dock)
- Smooth positioning while navigating Shorts
- Quick controls in the popup

**Install (Developer Mode)**
1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click "Load unpacked" and select this folder.

**Usage**
1. Open any Shorts video.
2. Click the extension icon to adjust the dock.
3. If changes don't apply instantly, refresh the Shorts page (F5).

**Notes**
- The dock position stays fixed while you scroll; it only resizes to fit new content.
- Anchoring to the player is always enabled to avoid layout glitches.
